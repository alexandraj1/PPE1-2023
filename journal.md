Commandes :
mkdir => Créer un dossier
mv => Déplacer
cd => Changer de répertoire 
rm => Supprimer sous-dossier
Créer des sous-dossiers dans un dossier =>
for i in 1 2 3; do
  mkdir sous_dossier$i
done
